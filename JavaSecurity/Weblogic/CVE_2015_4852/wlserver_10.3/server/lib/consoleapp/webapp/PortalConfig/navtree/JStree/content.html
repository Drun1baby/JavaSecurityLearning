Features
  • JavaScript Tree - object oriented (as much as makes sense)
  • Tree API for adding nodes, removing nodes, walking the tree
  • Custom Icons and links for each node
  • - Missing Connector Bars |_ etc..
  • Supports deferred loading
  • This version is built more for cross-browser compatibility
